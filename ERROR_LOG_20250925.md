- ユーザー体験の低下

**関連ファイル**:

- franchise-register/dist/js/ai-assistant.js
- GAS側: searchCompany関数

**具体的なエラー箇所**:
```javascript
// ai-assistant.js Line 212-223
script.src = fullUrl;

    console.error('JSONP読み込みエラー - URL:', script.src);
    // ここでエラーが発生
};
```

**考えられる原因**:
- [x] JSONP通信エラー（確定）

- GAS側でJSONPコールバックが正しく返されていない

- GAS側のsearchCompany関数のレスポンス形式エラー

**対応予定**:
- [ ] GAS側でJSONPレスポンスが正しく返されているか確認
