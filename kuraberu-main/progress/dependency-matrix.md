# 外壁塗装くらべる LP 依存関係マトリクス

このファイルは外壁塗装くらべるLPの各ファイル間の依存関係を記録しています。

## 依存関係の種類

- **依存**: 行のファイルが列のファイルに依存している
- **被依存**: 行のファイルが列のファイルから依存されている
- **独立**: 相互に依存関係がない
- **-**: 同一ファイル
