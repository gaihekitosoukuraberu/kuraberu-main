# 見積もりキープシステム

## ファイル構造
```
estimate-keep-system/
├── index.html         # メインHTMLファイル（キープ機能付き）
├── images/           # 画像ファイル
├── js/              # JavaScriptファイル
└── README.md        # このファイル
```

## 主な機能
1. **キープ機能**: 業者をキープして後でまとめて見積もり依頼可能
2. **見積もりボタン**: 「もっと見る」ボタンの下に配置
3. **レスポンシブ対応**: モバイル・デスクトップ両対応

## 最近の変更
- 見積もりボタンを固定位置から「もっと見る」の下に移動
- ランキングセクションに適切な余白を追加
- 新しい整理されたフォルダ構造に移行

## 使用方法
1. ブラウザで `index.html` を開く
2. 郵便番号を入力して検索
3. 業者をキープ
4. キープした業者をまとめて見積もり依頼
