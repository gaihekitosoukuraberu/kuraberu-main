## 🚀 GASデプロイ手順

### 1. Google Apps Scriptプロジェクト作成
1. Google Driveで新規 > その他 > Google Apps Script
2. プロジェクト名を「請求財務管理システム」に変更

### 2. コード設置
1. `gas-billing-finance.gs`の内容をコピー
2. Apps Scriptエディタに貼り付け
3. ファイル > 保存

### 3. スプレッドシートID設定
```javascript
const SPREADSHEET_ID = 'YOUR_SPREADSHEET_ID'; // ここを実際のIDに置換
```
スプレッドシートのURLから取得:
`https://docs.google.com/spreadsheets/d/【ここがID】/edit`

### 4. スクリプトプロパティ設定（オプション）
プロジェクト設定 > スクリプトプロパティ
- `FREEE_API_KEY`: freee APIキー（取得方法は下記）
- `FREEE_COMPANY_ID`: freee事業所ID

### 5. Webアプリとしてデプロイ
1. デプロイ > 新しいデプロイ
2. 種類: ウェブアプリ
3. 説明: v1.0（任意）
4. 実行ユーザー: 自分
5. アクセス可能なユーザー: 全員
6. デプロイ

### 6. デプロイURLを取得
デプロイ完了後に表示されるWebアプリのURLをコピー

## 🔧 トラブルシューティング

### エラー: CORS
GASデプロイ設定で「全員」アクセス可能になっているか確認

### エラー: 認証失敗
スクリプトプロパティのAPIキーが正しいか確認

### データが表示されない
1. スプレッドシートのシート名が正しいか確認
2. データが入力されているか確認
3. 日付フィルタが適切か確認

## 📝 注意事項

1. **税率変更時**: `BILLING_CONFIG.taxRate`を更新
2. **支払日変更時**: `BILLING_CONFIG.paymentDates`を更新
3. **定期バックアップ**: スプレッドシートの定期的なバックアップを推奨
4. **アクセス権限**: スプレッドシートの共有設定に注意

## 🆘 サポート

問題が発生した場合は、以下を確認してください:
- GASの実行ログ（表示 > ログ）
- ブラウザのコンソールログ
- スプレッドシートのデータ整合性
